GS1 Variant Selector (Stock)

This module enables scanning of GS1 barcodes (AIs 00, 01, 02, 10, 15, 17, 37) and, when a GTIN is shared across multiple variants, opens a variant selector to choose the correct product.

  • Template-first GTIN lookup via x_shared_barcode
  • Variant selection client action (tag='open_variant_selector')
  • Lot/expiry handling from GS1 AIs (10, 15/17)
